Thema Wordpress

The assignment was to create a custom theme in Wordpress and use PHP. Because the focus was on Wordpress and PHP, this had to be done in an existing HTML/CSS file.

Proces

In the previous block I created a portfolio website using Bootstrap. Because this website was the most complete when it comes to pages and potential content I chose to rebuild this website. During the lessons we were given an explanation about Wordpress and PHP. Here it was explained how to use Wordpress and how to set up your own theme. We started with an existing child theme to explore what a PHP file looks like exactly and what the basis is. The goal was to create a blog.

coding | programming

In principle, the lessons gave me enough information to get started with the theme. For example, the navigation section is based on the menu options in Wordpress, and I could convert this to PHP myself. I could also add the correct PHPP code to function.php. However, I did notice that I sometimes did not understand it or wanted things that were not discussed. The internet was my best friend. For this subject, I spent many hours on YouTube. The tutorials and walk-throughs on YouTube helped me to understand how PHP works and why it works.

Reflection

I thought it was a very nice assignment in which I laid a foundation for the use of Wordpress and PHP. In the old design I had created a styling that could have been applied to the post pages. I had not done this here, but it would have been a nice challenge. Furthermore, I could have adjusted the styling of the text in the post in CSS. Now the text is very close to the frames. In the end I am very satisfied and I really have the idea that I have learned something from this that I can use later.
